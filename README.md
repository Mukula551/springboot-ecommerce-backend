# Spring Boot E-Commerce Backend

A production-style backend e-commerce application developed using Java, Spring Boot, REST APIs, Spring Data JPA, and MySQL/H2 database.

## Features

- Category Management APIs
- Product Management APIs
- RESTful API Architecture
- Spring Data JPA Integration
- Layered Architecture
- Maven Project Structure

## Technologies Used

- Java 17
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- H2 Database
- MySQL (Upcoming)
- REST APIs

## Project Structure

- Controller Layer
- Service Layer
- Repository Layer
- Entity Models

## Future Improvements

- JWT Authentication
- Spring Security
- Pagination & Sorting
- Exception Handling
- DTO Layer
- Swagger Documentation
